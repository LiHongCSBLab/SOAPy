SOAPy_st.tl.infiltration_analysis

SOAPy_st.tl.infiltration_analysis(adata: AnnData, cluster_key: str = 'clusters', sample_key: str | None = None, sample: str | int | list | None = None, inplace: bool = True) → AnnData

The infiltration score was calculated by the infiltration of non-parenchymal cells into parenchymal cells.

Parameters

adataanndata.AnnData

An AnnData object containing spatial omics data and spatial information.

cluster_keystr, optional

The label of cluster in adata.obs.

sample_keystr, optional

The keyword of sample id in adata.obs.columns.

sample: Union[int, str, list, None],

The samples involved in calculating infiltration score.

inplacebool, optional

Whether to change the original adata.

Returns

  • anndata.AnnData.uns ['SOAPy']['infiltration score']['dic_crd_poly'] - infiltration score
